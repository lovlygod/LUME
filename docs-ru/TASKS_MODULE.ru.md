## Краткий Обзор

Модуль Tasks предоставляет управление задачами в стиле Kanban внутри проектов. Команды могут создавать, назначать, отслеживать задачи и комментировать их с четким рабочим процессом от создания до завершения.

### Workflow Статусов Задач

```
todo → in_progress → review → done
```

### Уровни Приоритета

- **low** - Желательно, не срочно
- **medium** - Стандартный приоритет
- **high** - Важно, должно быть сделано скоро
- **urgent** - Критично, требует немедленного внимания

### API Endpoints

- `POST /projects/:projectId/tasks` - Создать задачу
- `GET /projects/:projectId/tasks` - Получить все задачи проекта
- `PATCH /tasks/:taskId` - Обновить задачу
- `DELETE /tasks/:taskId` - Удалить задачу
- `POST /tasks/:taskId/comments` - Добавить комментарий
