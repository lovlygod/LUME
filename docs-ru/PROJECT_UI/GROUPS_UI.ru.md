## Обзор

Группы и каналы представлены внутри мессенджера как типы чатов. Публичные каналы доступны в поиске и поддерживают подписку; группы поддерживают заявки на вступление и управление участниками.

**Файлы:**
- `src/pages/Messages.tsx`
- `src/pages/messages/MessagesPage.tsx`
- `src/pages/messages/components/`
- `src/pages/messages/hooks/`

---

## Основные блоки

- Поиск публичных каналов
- Чат-экран группы/канала
- Роли и доступ (канал — read-only для не admin)
